# `@lattice/web`

Next.js App Router app. Scaffolding follows **`create-next-app`** (TypeScript, ESLint, no Tailwind).

- **Ports:** dev and production start use **3001** so **`@lattice/api`** can use **3000**.
- **Imports:** use **`@client/...`** for everything under `client/` (see `AGENTS.md`).

From the repository root:

```bash
npm run dev -w @lattice/web
```
