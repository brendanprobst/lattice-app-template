# `@lattice/web`

Next.js App Router app (static export). **Tailwind CSS v4**, **shadcn/ui** (Base UI), **lucide-react**.

- **Ports:** dev and production start use **3001** so **`@lattice/api`** can use **3000**.
- **Imports:** **`@client/pages/...`**, **`@client/components/ui/...`**, **`@client/lib/...`** (see [`AGENTS.md`](AGENTS.md)).
- **UI docs:** [`docs/ui-and-styling.md`](docs/ui-and-styling.md) — tokens, adding components, **`/`** and **`/ui`** pages.

From the repository root:

```bash
npm run dev -w @lattice/web
```
